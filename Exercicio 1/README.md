## Como executar (Linux)

### Iniciar o servidor
1. Abra o terminal no diretório do projeto e Execute:
- python3 server.py


### Iniciar o Cliente 

1. Baixe o projeto ou clone o repositório no computador do cliente.
2. Abra o terminal dentro da pasta onde está o arquivo client.py.
3. Execute:
- python3 client.py
