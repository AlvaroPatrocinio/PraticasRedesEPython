## **Descrição do Projeto**

Este repositório contém a implementação das atividades práticas do **Trabalho 2** da disciplina **Redes de Computadores 2**, ministrada pelo professor **Alessandro Vivas Andrade**.

Os códigos estão distribuídos em diretórios separados por exercício, conforme as orientações da atividade.

## **Introdução**

Este projeto apresenta o desenvolvimento do **Trabalho 2 da disciplina Redes de Computadores 2**, realizado por **Álvaro Patrocínio** e **João Pinheiro**.
O principal objetivo foi aplicar conceitos essenciais da comunicação em redes utilizando:

* TCP
* UDP
* ICMP
* DNS
* DHCP
* HTTP
* WebSockets

Além disso, foram utilizadas ferramentas de análise como o **Wireshark**, essenciais para inspecionar pacotes e compreender o funcionamento interno dos protocolos.

Os repositórios utilizados para organizar os códigos foram:

* **João Pinheiro**: github.com/joaopinheros/REDES_2
* **Álvaro Patrocínio**: github.com/AlvaroPatrocinio/PraticasRedesEPython

Os temas explorados incluem:

* Implementação de clientes e servidores TCP e UDP
* Uso de **threads** para conexões simultâneas
* Criação de um chat bidirecional via TCP
* Implementação de um servidor de hora multithread
* Chat com WebSockets
* Capturas e análises usando Wireshark
* Validações, tratamento de erros e boas práticas de programação
