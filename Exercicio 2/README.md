## Como executar (Linux)

1. Iniciar o servidor

### No computador onde o servidor vai rodar:

python3 servidor.py

O servidor ficará disponível para requisições.

2. Iniciar o cliente

### Em cada computador que vai entrar no chat:

* Baixe o projeto ou clone o repositório no computador do cliente.
* Abra o terminal dentro da pasta onde está o arquivo cliente.py.

Execute:

python3 cliente.py

## Comandos disponíveis no chat

* Digite qualquer mensagem para enviar ao outro cliente.
