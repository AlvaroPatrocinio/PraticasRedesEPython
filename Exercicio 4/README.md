## Como executar (Linux)
1. Iniciar o servidor

### No computador onde o servidor vai rodar:

python3 server_time.py

2. Iniciar o cliente

### No computador onde o cliente vai rodar:

- Baixe o projeto ou clone o repositório no computador do cliente.

- Abra o terminal dentro da pasta onde está o arquivo client_time.py.

### Execute:

- python3 client_time.py
