## Como executar (Linux)
1. Iniciar o servidor

### No computador onde o servidor vai rodar:

python3 server_chat.py

O servidor ficará esperando exatamente 2 clientes se conectarem.

2. Iniciar o cliente

### Em cada computador que vai entrar no chat:

- Baixe o projeto ou clone o repositório no computador do cliente.

- Abra o terminal dentro da pasta onde está o arquivo client_chat.py.

Execute:

python3 client_chat.py
